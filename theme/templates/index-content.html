Hi, there.

My name is Kenneth Reitz.

I'm an open source software developer that loves streamlining mundane things. I see beauty in elegant architectures. I try to write the simplest APIs possible, and I do so with Python.

The US Library of Congress, Twitter, Digg, Northrop Gruman, National Geographic, and Discovery Channel use my open source software.

I spend my days at NetApp, Inc writing internal enterprise tools and applications. My nights are spent passionately contributing to open source projects and hacking on what strikes my fancy. I also dabble in digital photography.

I occasionally cohost The Changelog and write a weekly blog post for The GitHub Blog.
